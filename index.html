Puppet

A configuration managment system

Nodes

How you associate puppet code to hosts

Examples


node 'testnode.test.com' {
	include foo
}

node /test\d\d.test.com/ {
	include bar
}

Resources

The building blocks of a system

Defined in Ruby or in the Puppet Language

Examples


file{'/tmp/test':
	ensure  => 'file',
	owner   => 'root',
	group   => 'root',
	mode    => '0755',
	content => 'Hello World',
}

package{['tmux','tcpdump']:
	ensure => 'latest',
}

Variables


$service_name = 'httpd'
